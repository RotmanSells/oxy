## Фаза 11. Мультиязычность (Bilingual → Trilingual)

### 11.1 Создание translations.js
- Полный словарь с 160+ ключами
- 3 языка: RU / EN / HY

### 11.2 Переключатель языка
- **Было:** `<a>`-ссылки (RU / EN / HY)
- **Стало:** `<button>` с `data-lang` (RU / EN / HY)
- CSS: `.lang-btn` вместо `.lang-link`
- Активный язык подсвечивается

### 11.3 data-i18n атрибуты
Добавлены 165+ атрибутов по всему HTML:
- Все section-label, section-title, section-desc
- Hero: locations, title, desc, formula, CTA, stats
- About: lead, paragraphs, stats
- Projects: tags, names, locations, slider buttons
- AG Club: labels, metrics, note
- Services: 6 titles + 6 descriptions + links
- Supply: 3 titles + 3 descriptions
- Advantages: 6 titles + 6 descriptions
- Tourism: 3 titles + 3 descriptions
- Partners: 6 names
- Contacts: labels, placeholders, options
- Footer: titles, address, social
- Form: labels, placeholders, submit, success

### 11.4 Placeholder-переводы
`data-i18n-placeholder` на всех input и textarea.

### 11.5 Aria-переводы
`data-i18n-aria` для accessibility:
- «Открыть меню» / «Закрыть»

### 11.6 Сохранение языка
`localStorage.setItem('siteLang', lang)` — при перезагрузке тот же язык.

### 11.7 Armenian (HY)
